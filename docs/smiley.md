# Smileys and Emojis

## Smileys

The following smileys are automatically converted to emojis:

| Smiley | Alternative | Emoji |
|--------|-------------|-------|
| :-)    | :)          | 🙂    |
| :-D    | :D          | 😃    |
| :-DD   | :DD         | 😄    |
| :-(    | :(          | 🙁    |
| :-O    |             | 😮    |
| :-*    |             | 😘    |
| ;-)    | ;)          | 😉    |
| ;-(    | ;(          | 😥    |
| :-P    |             | 😛    |
| :-/    |             | 😕    |
| O:-)   | O:)         | 😇    |
| >:-)   | >:)         | 😈    |
| B-)    |             | 😎    |
| <3     |             | 💖    |
| :o)    |             | 🤡    |
