# General

## Theme

Set a theme for the app, either a light one or a dark one depending on your preferences, eyesight, screen and so on.

## System

### Automatically check for updates

This checks the GitHub API of the Xeres project once a day to see if there's a new release.

#### Technicalities

A connection is performed to `https://api.github.com/repos/zapek/Xeres/releases/latest` using a generic *Chrome on Windows* user-agent to avoid tracking.

The version gathered is compared locally and, if there's a newer release, an option to download it is offered. If accepted, the file + its signature is downloaded from GitHub using the same mechanism. Before installation, a signature verification is performed using the built-in public key from the running application.
If everything is fine, the update can be installed.

### Launch on system startup

Automatically launches Xeres when the user logs in. This is currently only implemented on a normal Windows installation.
