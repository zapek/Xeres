# Notifications

Shows a system notification for the following events:

### Show connections

Whenever a friend connection is established.

### Show broadcasts

Whenever a friend sends a broadcast message, which is a message sent to every one of its friends. These messages are usually sent for important events, like announcing a durable downtime or a problem.

Unfortunately it can also be abused so it is desirable sometimes to deactivate them.

### Show discovery

Whenever someone is on the LAN and has LAN discovery enabled. You also need to have it turned on yourself.
