# Chats

This tab contains the chat rooms. These are multi-user rooms, a bit like [IRC](https://en.wikipedia.org/wiki/IRC).

The list of rooms appears in the listview on the left. Once you join one, you can chat in it, and you will make it available to your friends as well.

There are two kinds of chat rooms:

- public: no restrictions
- private: only visible to invited members

## Creation

Press the create chat room button on the top left, you will be able to supply the following fields:

- name: the name of the room. Chose it wisely as it cannot be changed afterwards
- the topic of the room: cannot be changed afterwards either
- visibility: public or private
- security: check if you want to avoid possible spam. Only identities signed by a profile will be allowed in. It's easier to trace them in case of abuse
