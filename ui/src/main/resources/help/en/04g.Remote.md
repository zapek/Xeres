# Remote

This setting enables to connect to a running Xeres instance using a distant client (either another Xeres in remote client mode or the [Android client](https://github.com/zapek/Xeres-Android/)).

The connection is encrypted (TLSv3 with a self-signed certificate). The Android client supports certificate pinning, which is highly recommended to avoid man in the middle attacks.

## Enabled

Turn the feature on or off.

## Port

The port to use for incoming connection. It will need to be accessible from your router.

## Set with UPNP

Use UPNP to route the port on your router. UPNP needs to be enabled in the Network settings.

## Username

This is the default username, it cannot be changed.

## Password

The password. While you can use an empty password (for testing or local connections) it is not recommended to do so. Please setup a strong password. One is generated by default.
