# Sound

This setting allows to produce sound on some events. Simply select a short sound file for each event.

Supported media formats:

- MP3
- MP4 (AAC)
- WAV
- AIFF

## Message received

When a direct or distant private message is received. While the window is in focus and active, no new sounds are produced.

## Highlight

When someone mentions you in a room chat.

## Friend connected

When a friend connects to your running instance.

## Download complete

When a download is complete.

## Call

When someone calls you with VoIP.
