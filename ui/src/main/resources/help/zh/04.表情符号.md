# 表情符号

可以使用别名快速显示一些表情符号。更推荐直接使用操作系统快捷键输入（例如 Windows 系统按 Win+. 键）。

### 常用表情

:grin​: :grin:

:rofl​: :rofl:

:yum​: :yum:

:blush​: :blush:

:rage​: :rage:

:scream​: :scream:

:cry​: :cry:

:sob​: :sob:

:sick​: :sick:

:poop​: :poop:

:muscle​: :muscle:

:wave​: :wave:

:eyes​: :eyes:

:zzz​: :zzz:

:fire​: :fire:

:heart​: :heart:

:boom​: :boom:

### 国家/地区

:cc​: （使用国家/地区的互联网域名代码，例如 ch 代表瑞士，fr 代表法国等）
